php-pear安装

发表于 2022-09-06 更新于 2022-03-25 分类于 PHP

安装步骤

  • 下载go-pear.phar下来，把文件复制到你要安装php版本的目录，然后解压再把文件移到php7.3.4nts的目录

    我拿php7.3.4nts来做示范

  • 执行

    下载地址：http://pear.php.net/go-pear.phar

关键事项

  • 注意事项：第一个一定是选择 local。
  • 第一次执行的时候一定要指定 php的路径，输入13弹出对话框选择PHP的目录。
  • 之后直接按 Enter 继续。完成！
