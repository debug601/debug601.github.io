VirtualBox安装Centos

发表于 2022-04-09 更新于 2022-04-16 分类于 centos

vbox安装centos

打开vbox虚拟机，点击新建

类型选择Linux 版本选择 Red Hat(64-bit) 因为centos是red hat的一个分支，点下一步

设置内存大小，我给了2G

装载光驱，来到存储这行，点击没有盘片

点击 选择虚拟盘，然后去到你放镜像文件的文件，选择centos镜像就行了

装载镜像成功的样子

点击启动虚拟机，进入centos安装界面，选择 Install CentOS Linux 7，回车

选择语言: 中文->简体中文

选择安装系统的磁盘位置

磁盘默认设置

选择需要安装的软件

选择GNOME桌面，点完成

要改一下时间和时区 要确保是上海并且时间跟你的电脑对的上

设置ROOT密码

输入密码，点完成即可 zhu1979

设置好ROOT密码，系统就会自动安装，等30分钟

安装成功后重启

点击未接受许可证

勾上同意，点完成，回到页面点完成配置

选择汉语，点前进

键盘布局选择英语-美国，点前进

隐私选择关闭，点前进

时区选择上海

在线账号，选择跳过

填写用户名称，点击前进

设置密码，点前进

安装完成

一定要创建快照

vbox centos安装增强功能

点击运行

他需要超级管理员权限才安装，我们输入密码，点认证

回车退出，再打开命令行输入reboot 重启

安装完成，重启系统生效。

使用命令

su 输入密码之后进入root权限
